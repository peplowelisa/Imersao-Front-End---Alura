# 📺 Vintage TV - Streaming Retrô

**Live Demo:** [Clique aqui para acessar o site](https://peplowelisa.github.io/Imersao-Front-End---Alura/)

📺 Sobre o Projeto:
Projeto desenvolvido durante a **Imersão Front-End da Alura**. A proposta foi criar uma interface de streaming funcional, baseada na Netflix. 

💡 A Evolução da Ideia:
Inicialmente, o objetivo era apenas criar uma interface baseada na estética de Cuphead, utilizando o Xicrinho e o Caneco como perfis de usuário. No entanto, durante o processo de desenvolvimento, o conceito amadureceu: decidi dar um passo adiante e tratar os personagens como se fossem indivíduos reais vivendo nas décadas de 50 e 60.

🎞️ Curadoria Baseada em Personalidade:
Essa mudança de perspectiva transformou o catálogo. Em vez de uma lista genérica, o conteúdo foi curado para refletir o que esses "moradores dos anos 50" estariam assistindo, respeitando a personalidade de cada um:

Perfil Xicrinho: Um catálogo voltado para a aventura, animações e produções que refletem o seu perfil impulsivo e bastante hiperativo.

Perfil Caneco: Uma seleção focada em clássicos e conteúdos que conversam com seu perfil mais "sério".

---

## ✨ Funcionalidades

- **Seleção de Perfil:** Interface imersiva para escolher entre os perfis "Xicrinho" e "Caneco".
- **Persistência de Dados:** Uso de `localStorage` para manter o perfil selecionado entre as páginas.
- **Catálogo Dinâmico:** O conteúdo da página de catálogo muda automaticamente dependendo do perfil escolhido.
- **Efeitos Visuais Vintage:** Filtros de grain (ruído), scan lines (linhas de tubo) e paleta monocromática via CSS.
- **Responsividade:** Design adaptado para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:** Animações complexas, filtros `grayscale`, `gradients` e efeitos de textura.
* **JavaScript (Vanilla):** Manipulação de DOM, lógica de perfis e gerenciamento de estado local.
* **Google Fonts:** Uso das fontes *Chango* e *Special Elite*.

## 🎨 Design

A interface foi pensada para ser puramente "Dark Mode", simulando o vidro escuro de uma TV antiga. O efeito de *scanlines* e o *grain* de película foram feitos inteiramente com CSS e SVG inline para garantir performance.
