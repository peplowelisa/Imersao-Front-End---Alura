# Imersao-Front-End---Alura
Vintage TV  | Projeto front-end de uma plataforma de streaming (feito durante a Imersão Front End da Alura) inspirado nos anos 50/60. A interface simula um televisor antigo com efeitos de grain, scan lines e estética P&amp;B. Possui seleção de perfis e catálogos dinâmicos baseados em cada utilizador. Desenvolvido com HTML, CSS e JS.
